## 🎓 Level 3 Term 2

### EEE 304 | Digital Electronics Laboratory
* **[Design and Implementation of FPGA-based Vending Machine using Verilog](https://github.com/Rafeed272/Design-and-Implementation-of-an-FPGA-Based-Vending-Machine-Using-Verilog)**
  * *Description:* A finite state machine (FSM) based vending machine implemented on an FPGA.
  * *Tools:* `Verilog` `Quartus` `FPGA`

### EEE 318 | Control Systems Laboratory
* **[Medical Waste Segregation Using Conveyor Belt](https://github.com/Rafeed272/Medical-Waste-Segregation-Using-Conveyor-Belt)**
  * *Description:* Automated system to categorize medical waste using sensors and control logic.
  * *Tools:* `Control Systems` `Arduino/Microcontroller` `Sensors`

### EEE 316 | Power Electronics Laboratory
* **[Multifunctional Solar Inverter for Home Appliances](https://github.com/Rafeed272/Multifunctional-Solar-Inverter/tree/main)**
  * *Description:* Design and simulation of an inverter system optimized for solar energy integration.
  * *Tools:* `Power Electronics` `MATLAB/Simulink` `Circuit Design`
